# Biblio FSSM

Site web d'une bibliothèque universitaire — projet front-end S2.

🔗 **Live :** https://aissarak8.github.io/project-web-S2/

---

## Pages

- `index.html` — accueil avec slider et stats animées
- `catalogue.html` — catalogue avec recherche et filtres
- `services.html` — services et horaires
- `reservation.html` — réservation de tables (zones A/B/C)
- `evenements.html` — événements à venir
- `contact.html` — formulaire de contact et FAQ

## Stack

HTML5 · CSS · JavaScript

## Lancer en local

```bash
git clone https://github.com/Aissarak8/project-web-S2.git
cd project-web-S2
# ouvrir index.html dans le navigateur
```
